Functions in the directory 'ext' are external functions that are not part of the tooling for the TUBS Road User Dataset 
but used for common tasks, such as reading XML-files. Hence, the author's specific licence agreements must be taken into account. 
The original licences are collected in this directory. 

The TUBS labeling tool uses the following external functions:

xml_read:           	by Jaroslaw Tuszynski
xml_write:          	by Jaroslaw Tuszynski
fit_ellipse:        	by Ohad Gal
maximize:           	by Oliver Woodford
buttondlg:          	by Dirk
enableDisableFig:   	by Yair Altman
lineSegmentsIntersect: 	by Murat Erdem
